Flex helpers

Despite of our use of Bootstrap 3.3.6 , we have some helpers to have the cool features that flexbox gives to developers. For this helpers we use a simple formula:
flex-property-modifier. Where flex- is the prefix, property- is the name of the property to apply and modifier is a modifier to apply the class depending on the device context. For practical purposes we are using the responsive modifiers proposed in bootstrap: xs, sm , md and lg. NOTE: If you want to apply a property for all the sizes use xs modifier

Alignment

.flex-start-[modifier]

Code Example
<div class="row flex-start-xs well"> <div class="alert alert-info col-xs-6"> start </div> </div>
